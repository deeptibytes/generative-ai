# Generative AI Projects

Welcome to the Generative AI Projects repository! This folder contains a collection of projects demonstrating various applications of generative AI, large language models, and prompt engineering using LangChain and other frameworks.

## Project Index

Navigate to any project by clicking on the links below:

| Project Name | Description | Link |
|---|---|---|
| **Basic Chatbot with Ollama** | A lightweight Q&A chatbot using the Gemma 2B open-source model with local inference | [basic_chatbot_with_ollama](./basic_chatbot_with_ollama) |
| **Enhanced Chatbot with Ollama** | An advanced Q&A chatbot with Mistral model featuring adjustable temperature and token controls | [enhanced_chatbot_with_ollama](./enhanced_chatbot_with_ollama) |
| **Chatbot with OpenAI** | A professional Q&A chatbot leveraging OpenAI's GPT models (GPT-4, GPT-4-Turbo, GPT-4o) | [chatbot_with_openai](./chatbot_with_openai) |
| **Basic Prompt** | Introductory examples of prompt engineering and basic LLM interactions | [basic-prompt](./basic-prompt) |
| **Prompt Chain** | Advanced prompt chaining techniques for complex multi-step LLM workflows | [prompt-chain](./prompt-chain) |
| **Document Chain** | Document processing and question-answering using LangChain document chains | [document-chain](./document-chain) |
| **Retriever Chain** | Semantic search and retrieval-augmented generation (RAG) implementations | [retriever-chain](./retriever-chain) |
| **Basic Chatbot** | Core chatbot implementation with fundamental LangChain patterns | [chatbot](./chatbot) |

## Technologies Used

### Core Frameworks
- **LangChain**: LLM orchestration and chaining
- **Streamlit**: Web application development
- **OpenAI API**: Commercial large language models

### Local Models
- **Ollama**: Local inference engine
- **Mistral**: Open-source language model
- **Gemma 2B**: Lightweight language model

### Utilities
- **Python-dotenv**: Environment variable management
- **LangSmith**: LLM tracing and debugging (optional)
